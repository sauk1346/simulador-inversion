📈 Simulador Matemático de Inversiones

Este sistema fue desarrollado como proyecto académico para aplicar conceptos de programación, bases de datos y cálculo diferencial en un simulador financiero interactivo.

🎯 Objetivo

Permitir al usuario simular el crecimiento de una inversión utilizando distintos modelos matemáticos, mostrando resultados, gráficos y análisis automáticos.

⚙ Funcionalidades

  • Interés Simple.
  • Interés Compuesto.
  • Interés Compuesto con aportes.
  • Cálculo de derivadas.
  • Cálculo de límites.
  • Gráficos interactivos.
  • Análisis automático.
  • Cálculo del capital real considerando inflación.
  • Comparación entre modelos.
  • Historial utilizando SQLite.
  • Metas de ahorro.

💻 Tecnologías utilizadas

  • Python
  • Flask
  • SQLite
  • SymPy
  • Plotly
  • Bootstrap 5
  • HTML5
  • CSS3
  • JavaScript

📚 Conceptos Matemáticos Aplicados

  • Funciones exponenciales.
  • Interés Simple.
  • Interés Compuesto.
  • Derivadas.
  • Límites.
  • Crecimiento exponencial.
  • Interpretación gráfica.
